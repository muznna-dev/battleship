# Battleship Game

A console-based Battleship game developed in C++.

## About

This project recreates the classic Battleship game in a terminal environment. It features grid management, user interaction, and game-state tracking.

## Features

* Two-player grid-based gameplay
* Ship placement and tracking
* Hit/miss detection
* Game state management

## Technologies Used

* C++

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/muznna-dev/battleship-game.git
cd battleship-game
```

2. Compile using g++:

```bash
g++ Battleship.cpp -o battleship
```

3. Run the executable:

```bash
./battleship
```
